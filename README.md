## Description

API para testes e estudo de infraestrutura na AWS com VPC, EC2 e ElasticCache.

## Installation

```bash
$ git clone git@github.com:PedroHenry-Santos/test-queue-nestjs.git
```

## Running the app

```bash
$ chmod +x ./setup.sh

$ yarn install

$ make up
```
